## Format Guide

**Phase statuses**: Use consistent markers:
- `✅ COMPLETE (Date)` — Done and verified
- `🔧 IN PROGRESS (X%)` — Active work
- `📋 PLANNING` — Documented but not started
- `🟡 PARTIAL` — Some sub-tasks complete
- `⏸️ DEFERRED` — Postponed with reason

**Update frequency**: At phase transitions and major milestones — not every commit.

**Cross-references**: This is the single source of truth for overall project status. INCOMPLETE_FEATURES.md tracks the detail. They must agree.
